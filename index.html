Features

Gamemaker Particle Visualizer is a tool where you can make Gamemaker particles and preview them in real time, without the need to recompile the game every time, which is one of the biggest struggles in game development.
This tool is aimed at beginners and only requires very basic logic on how particles work.

  • A rich editor that lets you customize and preview Gamemaker particles in real time.
  • Importing particles to all your projects is quick and easy.
  • Sprite support, all packed in the file itself so you'll never lose the textures.
  • Everything can be saved in special .GPV files for future use.
